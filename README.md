# Taniti Island – Interactive Prototype
## Overview

This project is an interactive prototype for a tourism website for Taniti Island (fictional location). The prototype was designed to create a simple, easy-to-navigate experience that helps users quickly find information about travel, activities, lodging, transportation, and local attractions.

The project was designed in Figma and embedded into a basic HTML page for viewing.

## Prototype

[Live Prototype](https://noelledeangelo.github.io/Taniti_Tourism_prototype/)

## Features
- Interactive clickable prototype
- Multi-page navigation flow
- Organized content for quick decision-making
- Clear visual hierarchy and navigation structure

## Tools Used
- Figma
- HTML
- GitHub Pages

## Notes
This is a design prototype, not a fully developed website. It demonstrates layout, structure, and user flow rather than full functionality.
